Using the Log Reviewer
  1. For Sigma logs, in Studio, navigate to the logs folder for the appropriate artifact and state
  2. Right-click on the SAS log file and select Download
  3. Select the file in the drop down on the home page of the Log Review app and click Review
  4. Filter by Issue Type in the dropdown or type in the 'Filter Log Line' box to filter by log line text
